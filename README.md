# 👾 Missão Vitamina (RPG em C++)

Um jogo de RPG textual interativo feito em **C++ puro**, criado para testar lógica de programação, manipulação de variáveis, estruturas condicionais e de repetição (`switch-case`, `if/else`, `loops`), além de entrega de entretenimento com humor e referências!

---

## 🎮 Sobre o Jogo

O objetivo é simples (ou não): **preparar a sua vitamina**. Porém, cada escolha pode te levar por caminhos absurdos, mortes trágicas, trocadilhos infames ou conquistas secretas!

* 📜 **+450 linhas** de código escrito do zero.
* 🏆 **Sistema de Conquistas (Achievements):** Várias conquistas para desbloquear ao longo da história.
* ⭐️ **Easter Egg de Platina:** Uma recompensa especial para quem conseguir desbloquear todas as conquistas!
* 🌐 **Multilingue:** Disponível em Português (`missao_vitamina.cpp`) e Inglês (`smoothie_mission.cpp`).

---

## 🕹️ Como Jogar

Você não precisa baixar nem compilar nada na sua máquina para jogar! 

1. Acesse o [Compilador Online (Programiz)](https://www.programiz.com/online-compiler/1BWOG2aDzFugu).
2. Clique no botão **Run** (Play).
3. Responda às opções no terminal digitando o **número** ou **letra** correspondente e pressionando `Enter`.
4. *Dica:* Lembre-se de rolar a tela do terminal para baixo para ver o desenrolar da história!

---

## 🛠️ Tecnologias Utilizadas

* **Linguagem:** C++
* **Conceitos aplicados:**
  * Estruturas de Decisão Encadeadas (`switch / case` e `if / else`)
  * Laços de Repetição (`for`)
  * Controle de Estados de Progresso via `string`
  * Lógica de Booleans para Verificação de Conquistas/Platina
